# Project-Programming-Advanced-Euromoon
Dit project hoort bij het vak "Programming Advanced". Het is een beheersysteem voor een fictieve treindienst "Euromoon". Code wordt in "Java" geschreven.

# Functionaliteiten
- Het beheren van passagiers.
- Het configureren van treinen op basis van wagons en types locomotieven.
- Het aanmaken van reizen.
- Het koppelen van treinen met reizen.
- Het verkopen van tickets aan passagiers.
- Het bewaren van lijsen passagiers voor een bepaalde reis.
